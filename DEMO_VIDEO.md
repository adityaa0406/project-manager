# 🎬 Demo Video Recording Guide

This guide will help you create an effective 2-5 minute demo video.

## 📹 Recording Setup

### Software Options
1. **Mac**: QuickTime Player, ScreenFlow, OBS Studio
2. **Windows**: OBS Studio, Camtasia, ScreenFlow
3. **Free & Simple**: [OBS Studio](https://obsproject.com/) (all platforms)

### Recording Settings
- **Resolution**: 1920x1080 (Full HD) or 1280x720 (HD)
- **Frame Rate**: 30 FPS
- **Audio**: Record system audio + microphone
- **Bitrate**: 5000 kbps (good quality)

## ✍️ Demo Script (3-4 minutes)

### Part 1: Introduction (30 seconds)
```
"Hi! This is Project Manager, a full-stack web application 
for team collaboration and project management. 

It features:
- User authentication with role-based access
- Project and team management
- Real-time task tracking with status updates
- Beautiful, responsive dashboard
- And much more!

Let me show you how it works."
```

### Part 2: Authentication (45 seconds)
Show these steps:
1. Show the login page
2. Demonstrate signup by creating a new account
3. Show the email validation (if applicable)
4. Login with the new account
5. Show the dashboard after login

### Part 3: Project Management (60 seconds)
Show these features:
1. **Create Project**
   - Click "New Project"
   - Fill in: Title, Description, Due Date
   - Create project
   - Show project appears in list

2. **Add Team Members**
   - Click on project
   - Add member functionality
   - Show members list

3. **Project Status**
   - Show status dropdown
   - Change status (Active → Completed)
   - Show update happens

### Part 4: Task Management (60 seconds)
Show these features:
1. **Create Task**
   - Click "New Task"
   - Fill: Title, Description, Priority, Due Date
   - Assign to member
   - Create task

2. **Task Tracking**
   - Show tasks in different columns
   - Move task from Todo → In Progress → Done
   - Show task details
   - Show priority indicators

3. **Dashboard Overview**
   - Show stats (total tasks, completed, overdue)
   - Show recent projects
   - Show task distribution

### Part 5: Features Showcase (60 seconds)
Quickly show:
1. **Dashboard**
   - Overview with stats
   - Recent projects
   - Task summary

2. **Teams Page**
   - Create team
   - Show team members
   - Show team details

3. **My Tasks**
   - Filter by status
   - Show overdue alerts
   - Show task details

### Part 6: Role-Based Access (30 seconds)
Optionally show:
1. Login with different user (Member role)
2. Show what they can/cannot do
3. Show access denied when trying admin action

### Part 7: Conclusion (15 seconds)
```
"That's the Project Manager application! 

Key highlights:
✅ Complete project lifecycle management
✅ Real-time collaboration features
✅ Secure role-based access control
✅ Beautiful, intuitive user interface
✅ Fully functional and deployed on Railway

Thank you for watching!"
```

## 🎥 Recording Tips

### Before Recording
1. **Clean up your screen**
   - Close unnecessary apps
   - Set appropriate zoom level
   - Ensure good lighting

2. **Prepare demo data**
   - Have demo accounts ready
   - Create sample projects/tasks
   - Or seed database with demo data

3. **Test everything**
   - Ensure app is running
   - Check internet connection
   - Verify audio/microphone

### During Recording
1. **Speak clearly**
   - Slow down your pace
   - Take pauses between actions
   - Explain what you're doing

2. **Move mouse deliberately**
   - Highlight what you're clicking
   - Don't move too fast
   - Point to important elements

3. **Show real functionality**
   - Create actual projects
   - Assign real tasks
   - Show real-time updates

4. **Use the app naturally**
   - Don't skip parts
   - Show loading states
   - Demonstrate actual workflows

### After Recording
1. **Edit if needed**
   - Trim beginning/end
   - Add titles or captions
   - Enhance audio if needed

2. **Export in good quality**
   - MP4 format recommended
   - 1080p resolution
   - H.264 codec

## 🎬 Pro Tips

1. **Practice first**
   - Do a rehearsal run
   - Time yourself (2-5 min requirement)
   - Get the flow right

2. **Add voiceover**
   - Record script separately
   - Mix with screen recording
   - Ensures clear audio

3. **Show quality**
   - Use high resolution
   - Good lighting
   - Professional appearance

4. **Include Call-to-Action**
   - End with GitHub repo link
   - Mention deployment URL
   - Encourage testing
